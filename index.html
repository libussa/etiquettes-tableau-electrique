Mode d'emploi

  1. Je choisi mon tableau
  2. Je choisi ma taille d'étiquette : ancien format 5 cm / nouveau format 3 cm de hauteur
  3. Je configure la taille et le nombres de modules par rangée
  4. J'ajoute tous les modules
  5. Je sauvegarde mon travail via le permalien
  6. J'imprime la page (Ctrl+P)
